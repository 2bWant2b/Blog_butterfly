# 矢量场和矢量线

可类比电场线，矢量线的切线方向就是该处矢量A→\overrightarrow{A}A 的方向。矢量线满足微分方程：

dxAx=dyAy=dzAz\frac{dx}{A_x} = \frac{dy}{A_y} = \frac{dz}{A_z} Ax​dx​=Ay​dy​=Az​dz​

由此可以求出矢量线方程簇
