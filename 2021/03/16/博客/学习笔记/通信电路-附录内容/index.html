# 基本公式

谐振角频率：ω0=1LC\omega_0=\frac{1}{\sqrt{LC}}ω0​=LC​1​，谐振频率：f0=12πLCf_0=\frac{1}{2\pi \sqrt{LC}}f0​=2πLC​1​
带宽：B=f0QB=\frac{f_0}{Q}B=Qf0​​
品质因数分为固有品质因数Q0Q_0Q0​ 和有载品质因数QeQ_eQe​（考虑源电阻和负载电阻）

  1. 串联谐振电路

Q0=ω0Lr=1ω0CrQe=ω0LR=1ω0CR\begin{array}{l} Q_0 =\frac{\omega_0 L}{r}=\frac{1}{\omega_0 C r} \\ Q_e =\frac{\omega_0 L}{R}=\frac{1}{\omega_0 C R} \end{array} Q0​=rω0​L​=ω0​Cr1​Qe​=Rω0​L​=ω0​CR1​​

其中R=r+Rs+RLR=r+R_s+R_LR=r+Rs​+RL​
2. 并联谐振电路

Q0=RPω0L=ω0CRPQe=Reω0L=ω0CRe\begin{array}{l} Q_0 =\frac{R_P}{\omega_0 L}=\omega_0 C R_P \\ Q_e =\frac{R_e}{\omega_0 L}=\omega_0 C R_e \end{array} Q0​=ω0​LRP​​=ω0​CRP​Qe​=ω0​LRe​​=ω0​CRe​​

其中Re=RP//Rs//RLR_e=R_P//R_s//R_LRe​=RP​//Rs​//RL​

若是串联谐振电路，当f=f0f=f_0f=f0​ 时呈阻性，且此时电路阻抗值最小；当f>f0f>f_0f>f0​ 时，电路呈感性；当f<f0f<f_0f<f0​ 时，电路呈容性
若是并联谐振电路，当f=f0f=f_0f=f0​ 时呈阻性，且此时电路阻抗值最大；当f>f0f>f_0f>f0​ 时，电路呈容性；当f<f0f<f_0f<f0​ 时，电路呈感性

  1. 电感分压电路

n=(W1+W2W2)Ri=n2R\begin{array}{l} n =\left( \frac{W_1+W_2}{W_2} \right) \\ R_i =n^2 R \end{array} n=(W2​W1​+W2​​)Ri​=n2R​

  1. 电容分压电路

n=(C1+C2C1)Ri=n2R\begin{array}{l} n =\left( \frac{C_1+C_2}{C_1} \right) \\ R_i =n^2 R \end{array} n=(C1​C1​+C2​​)Ri​=n2R​
