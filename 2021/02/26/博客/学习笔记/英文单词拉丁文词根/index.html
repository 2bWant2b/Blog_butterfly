# 说明

本文用于记录一些拉丁文词根，便于记忆或者推导单词。

自己对前缀后缀不特别区分了，而且合并了一些共同的部分

# 词根记录

radic,raci 表示 “根”
put 表示 “思考，估计，干净”
-ceed,-cede,-cess 表示 “走，移动”
con 表示 “共同”
gragh,gram 表示 “写，图”
bio-,bi - 表示 “生活，活着”
quir-,quer-,que-,quis - 表示 “寻求，询问”
satir - 表示讽刺的
multi-,mult - 表示 “很多”
-tude 表示 “… 的状态，品质，条件”
radio-,radi-,rad - 表示 “射线，辐射”
dox,dogma 表示 “观点”
para 表示 “一半”
ag-,act-,agi-,agen - 表示 “行动，驱动，引领”
oper-,opus 表示 “工作”
soli-,sol - 表示 “单独的”
civi-,civ-,cit - 表示 “大都市，合法的居民”
uni-,un - 表示 “一个，单个，数字 1”
form,forma-,format - 表示 “形状结构，外表，特殊的安排，发展，生产”
can-,cann - 表示 “芦苇，管”
pecu-,pecun - 表示 “家畜，私人财产；特别的”
alter,al 表示 “其他，他人”
pro-,por-,pur - 表示 “… 之前；代表；向前；根据”
clam-,clamat-,claim - 表示 “呼喊，说话”
fem-,femi - 表示 “女性”
dict-,dic - 表示 “说话，断言，公布，指出”
ceal - 表示 “藏，秘密”
megnet 表示 “磁石”
duro-,dur - 表示 “坚硬的，持久的”
super - 表示 “在… 之上”
vid-,vis-,vu-, 表示 “看”
spin - 表示 “刺，脊椎”
nav“船，水手”
migr-,migrat-“移动，迁移”
tox “毒”
fix-“固定”
cit-,citat-“说，刺激，唤起”
liber-“自由”
vacu-“空虚，消失”
-ant,-ants“某人，某事，某物”
leg,lex “与法律相关的”
pict-,paint-“画面，图片”
materi-“材料，物质的，木头”
fus-“流，融化，融合”
turb-” 挑起，干扰，动荡 “
contra-,contro-,contre-“反对，对立；冲突，不和”
audio-,aud-,audi-,audit-“听”
-orium,-oria,-ory “用来表演一个地方”
port-“拿，带”
demo-,dem-,demio-,-demic,-deme“人民，人们”
hosti-,host-“敌人”
cyn-,cur “狗”
uni-,un-“单个，一个；数字 1”
later“边，一侧，侧面”
mal-“坏的，错的”
concil“调和，调解”
merc-“货物，商品”
