在纯 markdown 语法和博客中写mermaid是有区别的， markdown 中是和写代码块一样，把种类写成 mermaid 就好。而在博客中是写上{% mermaid %}内容{% endmermaid %}，这个能在网页显示却不能在编辑器上显示。本文只记录常用语法，详情参照官方文档
