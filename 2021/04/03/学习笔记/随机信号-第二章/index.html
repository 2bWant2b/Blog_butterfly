# 随机过程基本概念

对于每一刻时间都会产生一组随机变量ξ\xiξ，对于确定的一个随机变量都会在时间轴上画出一条与时间相关的函数，这称为 样本函数 或 时间函数。如果随机变量和时间都是变量，那么就会产生 时间函数簇 ，这就是随机过程。见图理解：

# 概率分布

设随机过程X(t)X(t)X(t) 在任意nnn 个时刻t1,t2,⋯ ,tnt_1,t_2, \cdots , t_nt1​,t2​,⋯,tn​ 的取值构成nnn 维随机矢量[X(t1),X(t2),⋯ ,X(tn)][X(t_1),X(t_2), \cdots , X(t_n)][X(t1​),X(t2​),⋯,X(tn​)]，则可定义随机过程X(t)X(t)X(t) 的nnn 维分布函数：

FX(x1,x2,⋯ ,xn;t1,t2,⋯ ,tn)=P{X(t1)⩽x1,X(t2)⩽x2,⋯ ,X(tn)⩽xn}F_X(x_1,x_2,\cdots ,x_n ; t_1,t_2,\cdots ,t_n)=P\{X(t_1)\leqslant x_1,X(t_2)\leqslant x_2, \cdots ,X(t_n)\leqslant x_n\} FX​(x1​,x2​,⋯,xn​;t1​,t2​,⋯,tn​)=P{X(t1​)⩽x1​,X(t2​)⩽x2​,⋯,X(tn​)⩽xn​}

# 随机过程的数字特征

随机变量 的数字特征通常是一个 确定值 ，而 随机过程 的数字特征通常是一个 确定性函数 。我们的计算方法就是把时间ttt 当作一个常量，然后对随机变量进行计算。

# 数学期望

mX(t)=E[X(t)]=∫−∞+∞xf(x;t) dxm_X(t)=E[X(t)]=\int_{-\infty}^{+\infty} xf(x;t) \,dx mX​(t)=E[X(t)]=∫−∞+∞​xf(x;t)dx

# 均方值和方差

二阶原点矩称为随机过程的均方值：

ΨX2(t)=E[X2(t)]=∫−∞+∞x2fX(x;t) dx\Psi_X^2(t)=E[X^2(t)]=\int_{-\infty}^{+\infty}x^2 f_X(x;t) \,dx ΨX2​(t)=E[X2(t)]=∫−∞+∞​x2fX​(x;t)dx

二阶中心矩称为随机过程的方差：

σX2(t)=D[X(t)]=E[(X(t)−mX(t))2]\sigma_X^2 (t)=D[X(t)]=E[(X(t)-m_X(t))^2] σX2​(t)=D[X(t)]=E[(X(t)−mX​(t))2]

方差和均方值之间的关系：

σX2(t)=ΨX2(t)−mX(t)2\sigma_X^2 (t) = \Psi_X^2(t) - m_X(t)^2 σX2​(t)=ΨX2​(t)−mX​(t)2

标准差或均方差：

σX(t)=D[X(t)]\sigma_{X}(t) = \sqrt{D[X(t)]} σX​(t)=D[X(t)]​

# 自相关函数

自相关函数用来描述随机过程任意两个时刻的状态之间的内在联系，通常用RX(t1,t2)R_X(t_1,t_2)RX​(t1​,t2​)：

RX(t1,t2)=E[X(t1)X(t2)]=∫−∞+∞∫−∞+∞x1x2fX(x1,x2;t1,t2) dx1dx2R_X(t_1,t_2)=E[X(t_1)X(t_2)]=\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} x_1x_2f_X(x_1,x_2;t_1,t_2) \,dx_1 dx_2 RX​(t1​,t2​)=E[X(t1​)X(t2​)]=∫−∞+∞​∫−∞+∞​x1​x2​fX​(x1​,x2​;t1​,t2​)dx1​dx2​

若对于任意的t1,t2t_1,t_2t1​,t2​，均有RX(t1,t2)=0R_X(t_1,t_2)=0RX​(t1​,t2​)=0，则随机过程任意两个状态之间正交

# 自协方差函数

若用随机过程的两个不同时刻之间的二阶混合中心矩来定义相关函数，我们称之为自协方差，用CX(t1,t2)C_X(t_1,t_2)CX​(t1​,t2​) 表示：

CX(t1,t2)=E[(X(t1)−mX(t1))(X(t2)−mX(t2))]=∫−∞+∞∫−∞+∞[X(t1)−mX(t1)][X(t2)−mX(t2)] dx1dx2C_X(t_1,t_2)=E[(X(t_1)-m_X(t_1))(X(t_2)-m_X(t_2))]=\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} [X(t_1)-m_X(t_1)][X(t_2)-m_X(t_2)] \,dx_1dx_2 CX​(t1​,t2​)=E[(X(t1​)−mX​(t1​))(X(t2​)−mX​(t2​))]=∫−∞+∞​∫−∞+∞​[X(t1​)−mX​(t1​)][X(t2​)−mX​(t2​)]dx1​dx2​

若对于任意的t1,t2t_1,t_2t1​,t2​，均有CX(t1,t2)=0C_X(t_1,t_2)=0CX​(t1​,t2​)=0，则随机过程任意两个状态之间不相关

比较自协方差和自相关函数的关系：CX(t1,t2)=RX(t1,t2)−mX(t1)mX(t2)C_X(t_1,t_2)=R_X(t_1,t_2) -m_X(t_1)m_X(t_2)CX​(t1​,t2​)=RX​(t1​,t2​)−mX​(t1​)mX​(t2​)，当X(t)X(t)X(t) 零均值，自协方差函数与自相关函数相等

比较自协方差和方差的关系：令t1=t2=tt_1=t_2=tt1​=t2​=t，则CX(t1,t2)=CX(t,t)=D[X(t)]=σX2(t)C_X(t_1,t_2)=C_X(t,t)=D[X(t)]=\sigma_X^2(t)CX​(t1​,t2​)=CX​(t,t)=D[X(t)]=σX2​(t)

# 互相关函数

描述两个随机过程之间的关联程度：

RXY(t1,t2)=E[X(t1)Y(t2)]=∫−∞+∞∫−∞+∞xyfX(x,y;t1,t2) dxdyR_{XY}(t_1,t_2)=E[X(t_1)Y(t_2)]=\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} xyf_X(x,y;t_1,t_2) \,dx dy RXY​(t1​,t2​)=E[X(t1​)Y(t2​)]=∫−∞+∞​∫−∞+∞​xyfX​(x,y;t1​,t2​)dxdy

若对于任意的t1,t2t_1,t_2t1​,t2​，均有RXY(t1,t2)=0R_{XY}(t_1,t_2)=0RXY​(t1​,t2​)=0，则称X(t)X(t)X(t) 和Y(t)Y(t)Y(t) 正交，此时CXY(t!,t2)=−mX(t1)mY(t2)C_{XY}(t_!,t_2)=-m_X(t_1)m_Y(t_2)CXY​(t!​,t2​)=−mX​(t1​)mY​(t2​)

# 互协方差函数

CXY(t1,t2)=E[(X(t1)−mX(t1))(Y(t2)−mY(t2))]=RXY(t1,t2)−mX(t1)mY(t2)C_{XY}(t_1,t_2)=E[(X(t_1)-m_X(t_1))(Y(t_2)-m_Y(t_2))]=R_{XY}(t_1,t_2)-m_X(t_1)m_Y(t_2) CXY​(t1​,t2​)=E[(X(t1​)−mX​(t1​))(Y(t2​)−mY​(t2​))]=RXY​(t1​,t2​)−mX​(t1​)mY​(t2​)

若对于任意的t1,t2t_1,t_2t1​,t2​，均有CXY(t1,t2)=0C_{XY}(t_1,t_2)=0CXY​(t1​,t2​)=0，则称X(t)X(t)X(t) 和Y(t)Y(t)Y(t) 互不相关，此时RXY(t1,t2)=mX(t1)mY(t2)R_{XY}(t_1,t_2)=m_X(t_1)m_Y(t_2)RXY​(t1​,t2​)=mX​(t1​)mY​(t2​)

若X(t)X(t)X(t) 和Y(t)Y(t)Y(t) 相互独立，则RXY(t1,t2)=mX(t1)mY(t2)R_{XY}(t_1,t_2)=m_X(t_1)m_Y(t_2)RXY​(t1​,t2​)=mX​(t1​)mY​(t2​)，因此相互独立可以推出不相关，反之不成立。
