数学期望

mX=E[X]=∫−∞+∞xfX(x) dx(连续)m_X=E[X]=\int_{-\infty}^{+\infty} xf_X(x) \,dx \tag{连续} mX​=E[X]=∫−∞+∞​xfX​(x)dx(连续)

E[X]=∑i=1nxiP{X=xi}=∑i=1nxipi(离散)E[X]=\sum_{i = 1}^{n} x_i P\{X=x_i\} = \sum_{i=1}^{n}x_i p_i \tag{离散} E[X]=i=1∑n​xi​P{X=xi​}=i=1∑n​xi​pi​(离散)

性质1：
c为常数

E[cX]=cE[X]E[cX] = cE[X] E[cX]=cE[X]

性质2：

E[c]=cE[c] = c E[c]=c

性质3：
X_n是 任意的 随机变量

E[X1+X2+⋯+Xn]=E[X1]+E[X1]+⋯+E[Xn]E[X_1 + X_2 + \cdots + X_n] = E[X_1] + E[X_1] + \cdots + E[X_n] E[X1​+X2​+⋯+Xn​]=E[X1​]+E[X1​]+⋯+E[Xn​]

性质3：
X、Y是两个的 相互独立 的随机变量

E[XY]=E[X]E[Y]E[XY]=E[X]E[Y] E[XY]=E[X]E[Y]

方差

σX2=D[X]=E[(X−E[x])2]=∫−∞+∞(x−E[X])2fX(x) dx(连续)\sigma_X^2 = D[X] = E[(X - E[x])^2] = \int_{-\infty}^{+\infty} (x-E[X])^2 f_X (x) \,dx \tag{连续} σX2​=D[X]=E[(X−E[x])2]=∫−∞+∞​(x−E[X])2fX​(x)dx(连续)

σX2=D[X]=E[(X−E[x])2]=∑i=1+∞(xi−E[X])2⋅Pi(离散)\sigma_X^2 = D[X] = E[(X - E[x])^2] = \sum_{i=1}^{+\infty} (x_i - E[X])^2 \cdot P_i \tag{离散} σX2​=D[X]=E[(X−E[x])2]=i=1∑+∞​(xi​−E[X])2⋅Pi​(离散)

性质1：
c为常数

D[cX]=c2D[X]D[cX] = c^2 D[X] D[cX]=c2D[X]

性质2：

D[c]=D[c]D[c] = D[c] D[c]=D[c]

性质3：
X、Y是两个 **相互独立 **的随机变量

D[X±Y]=D[X]+D[Y]D[X1+X2+⋯+Xn]=D[X1]+D[X1]+⋯+D[Xn]D[X \pm Y] = D[X] + D[Y] \\ D[X_1 + X_2 + \cdots + X_n] = D[X_1] + D[X_1] + \cdots + D[X_n] D[X±Y]=D[X]+D[Y]D[X1​+X2​+⋯+Xn​]=D[X1​]+D[X1​]+⋯+D[Xn​]

矩

n阶原点矩定义为：

mn=E[Xn]n=1,2,⋯m_n = E[X^n] \qquad n=1,2, \cdots mn​=E[Xn]n=1,2,⋯

n阶中心矩定义为：

μn=E[(X−E[X])n]n=1,2,⋯\mu_n= E[(X-E[X])^n] \qquad n=1,2, \cdots μn​=E[(X−E[X])n]n=1,2,⋯

二阶联合原点矩：

m11=E[XY]=RXYm_{11} = E[XY] = R_{XY} m11​=E[XY]=RXY​

它又被称为X和Y的相关矩。

二阶联合中心矩：

σXY=CXY=E[(X−E[X])(Y−E[Y])]=E(XY)−E(X)E(Y)\sigma_{XY} = C_{XY} =E[(X-E[X])(Y-E[Y])] = E(XY) - E(X)E(Y) σXY​=CXY​=E[(X−E[X])(Y−E[Y])]=E(XY)−E(X)E(Y)

它又被称为X和Y的协方差

相关系数

ρXY=σXYD[X]D[Y]=σXYσXσY\rho_{XY} = \frac{\sigma_{XY}}{\sqrt{D[X]D[Y]}} = \frac{\sigma_{XY}}{\sigma_X \sigma_Y} ρXY​=D[X]D[Y]​σXY​​=σX​σY​σXY​​

独立，相关与正交

随机变量X与Y相互独立的充要条件：

fXY(x,y)=fX(x)fY(y)f_{XY}(x,y) = f_X(x) f_Y(y) fXY​(x,y)=fX​(x)fY​(y)

随机变量X与Y不相关的充要条件：

σXY=0ρXY=0\sigma_{XY} = 0 \\ \rho_{XY} = 0 σXY​=0ρXY​=0

随机变量X与Y正交的充要条件，相关矩为零：

E[XY]=0E[XY] = 0 E[XY]=0

相互独立一定不相关，不相关却不一定相互独立，而正交和这两没关系（在无其他条件下）
