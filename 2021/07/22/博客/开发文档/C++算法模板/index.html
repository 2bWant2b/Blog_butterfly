# 输入输出

scanf("%d%d", &n, &m) 有返回值，返回的是成功输入的个数 int 类型，当它值为 EOF 时表明输入结束，可用 ctrl+z 来结束输入

cin >> n >> m 有返回值，如果成功读取所有变量，则返回 true ，否则 false

# 处理无结束标记的输入

如输出其中最大值：

Sample Input:
2 4 6 8 9 123 56 7
Sample Output:
123

可以用下面两种方法：

while (cin >> n) {
	if (n > max) max = n;
}

while (scanf("%d", &n)) {
	if (n > max) max = n;
}

# 字符串函数

字符数组： char str[100] = 'Michael Jackson' 也可不声明数组大小

strcmp(str1, str2) 字符串比较函数，若二者相等则返回 0，若 str1 小于 str2（从第一个逐个比较，比较出一个就返回结果）则返回负数

strcpy(str, "You love me") 字符串拷贝函数，将后者拷贝给前者

strcat(str1, str2) 字符串拼接函数，将 str2 拼接到 str1 后面

strupr(str) 和 strlwr(str) 转换大小写

strlen(str) 返回字符串长度，不包括结尾的 '\0'

# 字符串输入输出

str[0] = 't' 直接修改字符串某个字符， str[5] = 0 等效于 str[5] = '\0' ，相当于字符串的结尾

scanf("%s", str) 输入字符串，不要用取地址符 &。scanf 读取到 空格 为止。 print("%s", str)

cin >> str 和 scanf 差不多，都要注意数组越界问题。 cout << str

如果要读取一整行字符串包括空格： cin.getline(str, sizeof(str)) ，也可以用 gets(char str[])

# 字符串用法

遍历一个字符串：

char str[100] = "test";
int len = strlen(str);
for (int i = 0; i < len; i++) {
    // 对str[i]的操作
}

或者利用字符串结尾的 ASCII 码是 0：

char str[100] = "test";
for (int i = 0; s[i]; i++) {
    // 对str[i]的操作
}

# 基本概念

int *p; p 是一个指针，变量 p 的类型是 int *

int *p = (int *) 40000 表明 p 指向地址 40000，地址 p 就是地址 40000

如下图所示， p 指针指向地址 40000， *p 是 40000 后 4 个字节的内存空间，向内存空间写入内容 5000 并赋值给 n

实际使用指针过程中不会赋给指针绝对地址，我们通常会让一个指针指向一个变量

# 指针的运算

  1. 两个同类型的指针变量可以比较大小，比较的就是地址的大小
  2. 两个同类型的指针变量可以相减： p1 - p2 = (地址p1 - 地址p2)/sizeof(T) 也就是表示可以存放多少的变量
  3. 指针变量加减一个整数的结果是一个指针，同样可进行 p++ p-- 操作

# sort

sort(数组名 + n1， 数组名 + n2) 将数组下标为 [n1, n2) 的元素从小到大排序，n2 不在排序区间内

int a[] = {15, 2, 4, 5, 6, 7};
sort(a, a+3);
//对15 2 4进行排序

从大到小排序： sort(数组名 + n1， 数组名 + n2， greater<int>())

# 二分查找

二分查找都用于已经排好序的数组上查找

binary_search(a, a + n, 值) 返回 bool 类型

查找下界： T * lower_bound(a+n1, a+n2, 值) 返回指针 T* p ， *p 是查找区间里下标最小的，大于等于值的元素，如果找不到 p 指向下标为 n2 的元素

查找上界： T * upper_bound(a+n1, a+n2, 值)

st.insert 添加元素， st.find 查找元素， st.erase 删除元素
